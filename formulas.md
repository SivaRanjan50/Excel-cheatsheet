## Top 20 Most Useful Shortcuts
```
Ctrl + S - Save
Ctrl + C / V / X - Copy/Paste/Cut
Ctrl + Z / Y - Undo/Redo
Ctrl + F - Find
Ctrl + H - Replace
Ctrl + Arrow - Navigate edges
Ctrl + Shift + Arrow - Select to edge
Ctrl + Space / Shift + Space - Select column/row
Ctrl + 1 - Format Cells
F2 - Edit cell
F4 - Toggle absolute reference
F9 - Calculate worksheets
Alt + = - AutoSum
Ctrl + T - Create Table
Ctrl + Page Up/Down - Switch sheets
Ctrl + Shift + L - Toggle filter
Ctrl + ; - Insert current date
Ctrl + Shift + : - Insert current time
Alt + Enter - New line in cell
Ctrl + ` - Show formulas
```

## Formula Efficiency Tips
```
Use Tables (Ctrl+T) for dynamic ranges
Named Ranges for readability
Avoid volatile functions (TODAY, NOW, OFFSET, INDIRECT) when possible
Use XLOOKUP instead of VLOOKUP in Excel 365
Prefer SUMIFS over multiple SUMIFs
Use IFERROR to handle errors gracefully
Array formulas for complex calculations
LET function (Excel 365) for cleaner formulas
```
## Data Cleanup Checklist
```
Remove duplicates
Trim extra spaces (TRIM function)
Convert text to proper case (PROPER function)
Fix date formats
Fill blank cells
Split text to columns
Validate data types
Check for errors (#N/A, #VALUE!)
```
